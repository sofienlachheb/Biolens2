# BioLens Pro (Vite + React)

هذا المشروع هو تحويل مباشر لملف HTML الأصلي إلى مشروع Vite + React.

## التشغيل

1) افتح المجلد ثم ثبّت الحزم:

```bash
npm install
```

2) شغّل السيرفر المحلي:

```bash
npm run dev
```

سيفتح الرابط الذي يظهر في الطرفية (عادة http://localhost:5173).

## ملاحظات مهمة

- تم استخدام Chart.js عبر npm (الحزمة: chart.js).
- ميزات البلوتوث تعتمد على Web Bluetooth وقد لا تعمل إلا في متصفحات/أجهزة تدعمها (مثل Chrome على Android).
- قياسات العلامات الحيوية هنا ما زالت تجريبية (محاكاة) كما في النسخة الأصلية.
